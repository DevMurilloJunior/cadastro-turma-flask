# 🎓 Cadastro da Turma

Projeto desenvolvido com **Python + Flask** para realizar o cadastro de alunos.

O sistema permite:
- Cadastrar alunos
- Exibir total de cadastrados
- Listar alunos
- Limpar lista

---

# 🚀 Como executar o projeto

## 1️⃣ Instalar o Flask

```bash
pip install flask
```

---

## 2️⃣ Executar o projeto

```bash
python app.py
```

---

## 3️⃣ Abrir no navegador

```bash
http://127.0.0.1:5000
```
